# GitHub Basic Practice

GitHub Basic ワークショップで使う、個人用の練習リポジトリです。

このリポジトリには、ブロックが盤面の底で止まらない **Falling Blocks** アプリが入っています。最初からバグを直さず、次の順番で作業してください。

1. Issue に目的と完了条件を書く
2. `main` から作業 Branch を作る
3. `app/falling-blocks/game.js` の `FLOOR_ROW` を修正する
4. Commit / Push して Pull Request を作る
5. ペアで Review してから Merge する
6. Merge 後に Issue、履歴、`git blame` を確認する

## 修正対象

```text
app/falling-blocks/game.js
```

正解を先に見ず、まず `app/falling-blocks/index.html` をブラウザで開いて不具合を確認してください。
